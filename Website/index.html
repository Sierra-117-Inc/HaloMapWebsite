Corbulo Academy

Hello and welcome to Corbulo Academy! The purpose of this website is to provide new players and veterans alike with a repository of Halo 3 maps and callouts that we find useful to help coordinate our teamplay. We have been lifelong friends that have been playing Halo 3 since 2007 and wanted a fun project to collaborate on that would combine practicing our coding skills as well as work on something that we both feel passionate about. Let us know if you have any feedback or requests for the site!
